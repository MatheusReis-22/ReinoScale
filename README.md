
  # ReinoScale

Sistema de Gestão de Escalas e Eventos da Igreja

## Descrição

O ReinoScale é um aplicativo mobile desenvolvido para auxiliar igrejas no gerenciamento de escalas ministeriais, eventos, membros e avisos.

O sistema centraliza informações importantes da igreja em um único ambiente, facilitando a organização das atividades e a comunicação entre os participantes.

## Funcionalidades

- Login de usuários
- Dashboard principal
- Consulta de escalas
- Cadastro de novas escalas
- Visualização de eventos
- Consulta de membros
- Visualização de avisos
- Perfil do usuário

## Tecnologias Utilizadas

- React Native
- JavaScript
- Expo Snack
- Expo Go

## Como Executar

1. Acessar o link do projeto no Expo Snack.
2. Abrir o aplicativo Expo Go.
3. Escanear o QR Code disponibilizado.
4. Aguardar o carregamento da aplicação.
5. Utilizar normalmente as funcionalidades do sistema.

## Estrutura do Sistema

O aplicativo é composto pelas seguintes telas:

- Login
- Dashboard
- Escalas
- Nova Escala
- Eventos
- Perfil
- Membros
- Avisos

## Requisitos Funcionais

RF-01: Permitir autenticação de usuários.

RF-02: Exibir dashboard principal.

RF-03: Consultar escalas ministeriais.

RF-04: Cadastrar novas escalas.

RF-05: Visualizar eventos.

RF-06: Consultar perfil do usuário.

RF-07: Consultar membros cadastrados.

RF-08: Visualizar avisos.

RF-09: Navegar entre as telas do sistema.

RF-10: Exibir mensagem de confirmação ao cadastrar uma escala.
