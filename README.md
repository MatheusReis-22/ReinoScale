# ReinoScale

Sistema de Gestão de Escalas e Eventos da Igreja

## Descrição

O ReinoScale é um aplicativo mobile desenvolvido para auxiliar igrejas no gerenciamento de escalas ministeriais, eventos, membros e avisos.

## Funcionalidades

- Login de usuários
- Dashboard principal
- Consulta de escalas
- Cadastro de escalas
- Visualização de eventos
- Perfil do usuário
- Consulta de membros
- Avisos da igreja

## Tecnologias Utilizadas

- React Native
- JavaScript
- Expo Snack
- Expo Go

## Como Executar

1. Acesse o link do projeto no Expo Snack.
2. Abra o aplicativo Expo Go.
3. Escaneie o QR Code disponibilizado.
4. Aguarde o carregamento da aplicação.
5. Utilize normalmente o sistema.
